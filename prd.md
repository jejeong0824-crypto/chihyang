# MVP PRD
## 감상평 기반 영화/책 취향 분석 사이트 — "취향"

---

## 1. 제품 개요

영화/책 감상평(텍스트)을 남기면 AI가 취향을 분석해주는 서비스. **별점 없이**, 글만으로 취향을 파악한다.

---

## 2. MVP 핵심 기능

### 2.1 인증
- Google 소셜 로그인 (Supabase Auth)
- 닉네임 설정

### 2.2 감상평 작성
- 영화/책 검색 (TMDB API, Google Books API)
- 자유 형식 텍스트 감상평 작성
- 태그 추가 (예: #몰입감, #여운)
- 공개/비공개 설정
- 수정/삭제

### 2.3 AI 취향 분석
- 감상평 **3개 이상** 시 분석 결과 생성 (Claude API)
- 분석 항목:
  - **취향 키워드** — 반복되는 선호 요소 (예: "복선 회수", "도덕적 딜레마")
  - **취향 요약** — 1~2문장 설명
  - **취향 유형** — 카테고리 (예: "서사 몰입형", "감성 여운형")
- 감상평 추가 시 자동 업데이트

### 2.4 친구
- 고유 코드로 친구 요청/수락/거절
- 친구 목록
- 친구와 취향 비교 (겹치는 키워드, 일치도 %)
- 친구 공개 감상평 피드

### 2.5 마이페이지
- 내 감상평 목록 (영화/책 필터)
- 취향 프로필 카드

## 3. 화면별 상세 User Flow

### 3.1 `/login` — 로그인

| 요소 | 내용 |
|------|------|
| 진입 | 비로그인 상태에서 모든 페이지 접근 시 리다이렉트 |
| 화면 구성 | 서비스 로고 + 한줄 소개 + Google 로그인 버튼 |
| 사용자 행동 | Google 로그인 클릭 |
| 분기 | 첫 가입 → 닉네임 설정 모달 → `/` 이동 |
| | 기존 회원 → `/` 이동 |

---

### 3.2 `/` — 홈

| 요소 | 내용 |
|------|------|
| 진입 | 로그인 완료 후 / 상단 로고 클릭 |
| 화면 구성 | 상단 네비게이션 (홈, 감상평 작성, 친구, 마이페이지) |
| | 친구 공개 감상평 피드 (최신순 타임라인) |
| | 친구가 없거나 피드가 비어있으면 → 빈 상태 안내 ("친구를 추가하고 감상평을 확인해보세요") |
| 사용자 행동 | 감상평 카드 클릭 → `/review/[id]` 이동 |
| | 작성자 닉네임 클릭 → `/friends/[id]` 이동 |
| | "감상평 쓰기" 버튼 → `/review/new` 이동 |

---

### 3.3 `/review/new` — 감상평 작성

| 요소 | 내용 |
|------|------|
| 진입 | 홈 "감상평 쓰기" 버튼 / 네비게이션 |
| 화면 구성 | **Step 1** — 콘텐츠 검색 |
| | 영화/책 탭 전환 + 검색창 |
| | 검색 결과 리스트 (포스터/표지 + 제목 + 연도/저자) |
| | **Step 2** — 감상평 입력 |
| | 선택한 콘텐츠 정보 표시 (포스터 + 제목) |
| | 감상평 텍스트 입력 (placeholder: "이 작품에 대한 생각을 자유롭게 적어보세요") |
| | 태그 입력 (쉼표 또는 엔터로 구분, 자동 #접두사) |
| | 공개/비공개 토글 (기본: 공개) |
| | "저장" 버튼 |
| 사용자 행동 | 콘텐츠 선택 → 감상평 입력 → 태그 추가 → 저장 |
| 분기 | 저장 성공 → `/review/[id]` 이동 |
| | 감상평 3개째 저장 시 → 저장 후 "취향 분석이 생성되었어요!" 토스트 표시 |

---

### 3.4 `/review/[id]` — 감상평 상세

| 요소 | 내용 |
|------|------|
| 진입 | 홈 피드에서 카드 클릭 / 마이페이지에서 감상평 클릭 |
| 화면 구성 | 콘텐츠 정보 (포스터 + 제목 + 연도/저자) |
| | 감상평 본문 |
| | 태그 목록 |
| | 작성자 닉네임 + 작성일 |
| | 본인 작성일 경우: 수정/삭제 버튼 |
| 사용자 행동 | 수정 클릭 → 감상평 편집 모드 (인라인) → 저장 |
| | 삭제 클릭 → 확인 모달 → 삭제 후 `/my` 이동 |
| | 작성자 클릭 → `/friends/[id]` 이동 |

---

### 3.5 `/my` — 마이페이지

| 요소 | 내용 |
|------|------|
| 진입 | 네비게이션 "마이페이지" |
| 화면 구성 | **프로필 영역** — 프로필 이미지 + 닉네임 + 친구 코드 (복사 버튼) |
| | **취향 프로필 카드** (감상평 3개 이상일 때 표시) |
| | 취향 유형 뱃지 (예: "서사 몰입형") |
| | 취향 키워드 태그들 |
| | 취향 요약 문장 |
| | 감상평 3개 미만일 때 → "감상평을 n개 더 작성하면 취향을 분석해드려요" |
| | **내 감상평 목록** — 영화/책 필터 탭 + 감상평 카드 리스트 (최신순) |
| 사용자 행동 | 취향 카드 확인 |
| | 감상평 카드 클릭 → `/review/[id]` 이동 |
| | 친구 코드 복사 → 클립보드 복사 + 토스트 |

---

### 3.6 `/friends` — 친구 목록

| 요소 | 내용 |
|------|------|
| 진입 | 네비게이션 "친구" |
| 화면 구성 | **친구 추가 영역** — 친구 코드 입력창 + "요청 보내기" 버튼 |
| | **받은 요청** — 요청자 닉네임 + 수락/거절 버튼 (요청이 있을 때만 표시) |
| | **친구 목록** — 프로필 이미지 + 닉네임 + 취향 유형 뱃지 |
| | 친구가 없으면 → 빈 상태 안내 ("친구 코드를 공유해서 친구를 추가해보세요") |
| 사용자 행동 | 친구 코드 입력 → 요청 보내기 → 성공 토스트 |
| | 받은 요청 수락 → 친구 목록에 추가 |
| | 받은 요청 거절 → 요청 제거 |
| | 친구 카드 클릭 → `/friends/[id]` 이동 |

---

### 3.7 `/friends/[id]` — 친구 프로필 + 취향 비교

| 요소 | 내용 |
|------|------|
| 진입 | 친구 목록에서 클릭 / 홈 피드에서 닉네임 클릭 |
| 화면 구성 | **친구 프로필** — 프로필 이미지 + 닉네임 + 취향 유형 뱃지 |
| | **취향 비교** (둘 다 프로필이 있을 때) |
| | 나의 키워드 vs 친구 키워드 (겹치는 키워드 강조) |
| | 취향 일치도 (%) |
| | 나의 취향 요약 vs 친구 취향 요약 |
| | 한쪽이라도 프로필이 없으면 → "아직 취향 분석이 완료되지 않았어요" |
| | **친구 공개 감상평 목록** — 감상평 카드 리스트 |
| 사용자 행동 | 감상평 카드 클릭 → `/review/[id]` 이동 |

---

### 3.8 화면 이동 맵

```
/login ──(로그인)──→ /
                      │
         ┌────────────┼────────────┐
         ↓            ↓            ↓
    /review/new     /friends      /my
         │            │            │
         ↓            ↓            │
    /review/[id]  /friends/[id]   │
         ↑                         │
         └─────────────────────────┘
```

## 4. 기술 스택

| 영역 | 선택 |
|------|------|
| 프레임워크 | Next.js 16 (App Router, Turbopack) |
| 스타일링 | Tailwind CSS 4 + shadcn/ui (radix-lyra) |
| 다크모드 | next-themes |
| 인증 | Supabase Auth (Google OAuth) |
| DB | Supabase (PostgreSQL) |
| ORM | Prisma |
| AI | Claude API |
| 영화 검색 | TMDB API |
| 책 검색 | Google Books API |

---

## 5. 데이터 모델

| 엔티티 | 주요 필드 |
|--------|----------|
| **User** | id, email, nickname, profileImage, friendCode |
| **Review** | id, userId, contentType(movie/book), contentId, contentTitle, body, tags[], isPublic, createdAt |
| **TasteProfile** | id, userId, keywords[], summary, type, updatedAt |
| **Friendship** | id, requesterId, receiverId, status(pending/accepted/rejected), createdAt |

## 6. 페이지 구조

| 경로 | 설명 |
|------|------|
| `/` | 랜딩 (비로그인) / 홈 피드 (로그인) |
| `/login` | 로그인 |
| `/review/new` | 감상평 작성 |
| `/review/[id]` | 감상평 상세 |
| `/my` | 마이페이지 (감상평 + 취향 프로필) |
| `/friends` | 친구 목록 + 추가 |
| `/friends/[id]` | 친구 프로필 + 취향 비교 |

## 7. MVP 제외 (향후)

- 취향 기반 콘텐츠 추천
- 취향 프로필 이미지 내보내기
- Kakao 로그인
- 알림 시스템
- 댓글/반응
- 장르 선호 분포 차트
- 감상 통계 (월별, 태그별)
