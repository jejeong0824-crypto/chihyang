# User Flow — "취향"

---

## Flow 1. 첫 가입 사용자

```
1. 서비스 접속 (/)
   └─ 비로그인 → /login 리다이렉트

2. /login
   └─ Google 로그인 클릭
   └─ OAuth 인증 완료

3. 닉네임 설정 모달
   └─ 닉네임 입력 → "시작하기" 클릭

4. / (홈)
   └─ 친구 피드 비어있음
   └─ "감상평을 작성해보세요" 안내 표시
```

---

## Flow 2. 감상평 작성

```
1. / (홈) 또는 네비게이션
   └─ "감상평 쓰기" 클릭

2. /review/new — Step 1: 콘텐츠 검색
   └─ 영화/책 탭 선택
   └─ 검색어 입력 → 결과 리스트 표시
   └─ 콘텐츠 클릭하여 선택

3. /review/new — Step 2: 감상평 입력
   └─ 선택한 콘텐츠 정보 확인 (포스터 + 제목)
   └─ 감상평 텍스트 작성
   └─ 태그 추가 (선택)
   └─ 공개/비공개 설정
   └─ "저장" 클릭

4. /review/[id] — 감상평 상세
   └─ 저장된 감상평 확인

   [분기] 감상평이 3개째인 경우
   └─ "취향 분석이 생성되었어요!" 토스트 표시
```

---

## Flow 3. 취향 분석 확인

```
1. 네비게이션 → "마이페이지"

2. /my
   └─ 취향 프로필 카드 확인
       ├─ 취향 유형 뱃지 (예: "서사 몰입형")
       ├─ 취향 키워드 태그들
       └─ 취향 요약 문장

   [분기] 감상평 3개 미만
   └─ "감상평을 n개 더 작성하면 취향을 분석해드려요" 안내
```

---

## Flow 4. 감상평 수정/삭제

```
1. /my → 감상평 카드 클릭

2. /review/[id]
   └─ 본인 작성 감상평 → 수정/삭제 버튼 표시

   [수정]
   └─ "수정" 클릭 → 편집 모드 전환
   └─ 내용 수정 → "저장" 클릭
   └─ 수정 완료 확인

   [삭제]
   └─ "삭제" 클릭 → 확인 모달 표시
   └─ "삭제" 확인 → /my 이동
```

---

## Flow 5. 친구 추가

```
1. 네비게이션 → "친구"

2. /friends
   └─ 친구 코드 입력창에 상대방 코드 입력
   └─ "요청 보내기" 클릭
   └─ "친구 요청을 보냈어요" 토스트 표시

3. (상대방) /friends
   └─ "받은 요청" 영역에 요청 표시
   └─ "수락" 클릭 → 친구 목록에 추가
   └─ 또는 "거절" 클릭 → 요청 제거
```

---

## Flow 6. 친구 코드 공유

```
1. 네비게이션 → "마이페이지"

2. /my
   └─ 프로필 영역의 친구 코드 옆 복사 버튼 클릭
   └─ 클립보드에 복사 완료
   └─ "복사되었어요" 토스트 표시
   └─ 메신저 등으로 친구에게 전달
```

---

## Flow 7. 취향 비교

```
1. /friends → 친구 카드 클릭

2. /friends/[id]
   └─ 친구 프로필 확인 (닉네임 + 취향 유형)
   └─ 취향 비교 영역 확인
       ├─ 나의 키워드 vs 친구 키워드 (겹치는 키워드 강조)
       ├─ 취향 일치도 (%)
       └─ 나의 요약 vs 친구 요약

   [분기] 한쪽이라도 취향 분석 미완료
   └─ "아직 취향 분석이 완료되지 않았어요" 안내
```

---

## Flow 8. 친구 감상평 탐색

```
1. / (홈)
   └─ 친구 공개 감상평 피드 (최신순)
   └─ 감상평 카드 클릭 → /review/[id]
   └─ 작성자 닉네임 클릭 → /friends/[id]

   또는

1. /friends → 친구 카드 클릭

2. /friends/[id]
   └─ 하단 친구 공개 감상평 목록
   └─ 감상평 카드 클릭 → /review/[id]
```

---

## Flow 9. 기존 회원 재방문

```
1. 서비스 접속 (/)
   └─ 로그인 세션 유지 → / (홈) 바로 진입

2. / (홈)
   └─ 친구 감상평 피드 확인
   └─ 새 감상평 작성 또는 마이페이지 확인
```
